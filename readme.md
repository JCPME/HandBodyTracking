# Foobar

This is a python repo for hand and bodytracking using mediapipe and orbecsdk

## Installation

1. Install the pyorbecsdk


```bash
pip install pyorbbecsdk-2.0.15-cp312-cp312-linux_x86_64.whl
```
2. Install the requirements

```
pip install -r requirements.txt
```

## Usage

```
python3 handtracking.py
```
